my dotfiles
=========

A simple repo to backup, sync, and show off some simple configurations. Inspired by [Drew DeVault's method](https://drewdevault.com/2019/12/30/dotfiles.html) of dotfile management.

Example dotfiles for:

  - `vim`
  - `git`
  - `mintty` / `git bash`
  - `bash aliases`
  - `gnu screen`
  - `starship`
  - that's it!
